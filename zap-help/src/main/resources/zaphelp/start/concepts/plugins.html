Plugins

The Paros Proxy supports software plugins which can add additional functionality.
Plugins written for Paros may still work with ZAP.

Plugins can add extra top level and right click menu items.
They can also add functionality to the active scanner.
Scanner plugins can be enabled and disabled via the Scan Policy dialog

The intention is to continue to support Paros plugins, although it is possible that future changes may prevent some plugins from working.

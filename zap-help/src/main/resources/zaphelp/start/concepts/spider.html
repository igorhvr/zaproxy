Spider

The spider scans the responses received and reports any URLs that have not yet been visited.

It is recommended that you manually explore the application first and only use the spider to find links that you have missed or are hidden in some way.

The spider is configured using the Options Spider screen.

Accessed via

     Spider tab
     Sites tab'Attack/Spider site' right click menu item

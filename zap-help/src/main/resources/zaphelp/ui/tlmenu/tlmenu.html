Top Level Menu

By default the following top level menu items will be present:
     File menu This handles the current session
     Edit menu This handles searching for strings and managing session tracking.
     View menu This handles the display options
     Analysis menu This handles spidering and automated scanning
     Report menu This handles the reports
     Tools menu This handles the tools and general options
     Help menu This gives access to the 'about' dialog and this help file

Note that plugins can add additional menu items.

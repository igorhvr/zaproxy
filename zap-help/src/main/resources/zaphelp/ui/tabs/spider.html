Spider tab

The Spider tab shows you the URIs found by the spider that you have not yet visited.
Sites can be selected via the toolbar or the Sites tab.
In this release the spider can only be run against one site at a time.
The toolbar provides a set of buttons which allow you to start, stop, pause and resume the scan.
A progress bar shows how far the scan of the selected site has progressed.

Right click menu

Right clicking on a node will bring up a menu which will allow you to:

Find...

This will bring up the Find dialog.

Encode/Decode/Hash...

This will bring up the Encode/Decode/Hash dialog.
If you have highlighted any text then this will be automatically included in the dialog.

Copy

This will copy the selected string to the clipboard.

Break Points tab

The Break Points tab shows you all of the break points that you have set.
Break points can be set via the History and Sites tabs.

Right click menu

Right clicking on a break point will bring up a menu which will allow you to:

Edit...

This will bring up the Edit Break Point dialog which allows you to edit the regex that defines the break point.

Remove

This will remove the break point

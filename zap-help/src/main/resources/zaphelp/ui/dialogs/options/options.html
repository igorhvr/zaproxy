Options dialog

The Options dialog allows you to configure ZAP.
It include the following screens:
     Active Scan
     Anti CSRF Tokens
     API
     Applications
     Authentication
     Brute Force
     Certificate
     Check for Updates
     Connection
     Display
     Dynamic SSL Certificate
     Fuzzer
     Local Proxy
     Passive Scan
     Port Scan
     Spider

Accessed via

     Top level Tools menu'Options ...' menu item
